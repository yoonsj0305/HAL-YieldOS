## Release title

HAL YieldOS v3.0.11 - GitHub Public PoC Baseline

## Short summary

HAL YieldOS is a read-only Functional Yield Evidence Layer for imperfect industrial systems.

It generates candidate-only evidence packs and functional passports for human review.

It does not control hardware, modify recipes, guarantee yield, certify root cause, or run Recovery Compiler.

## Highlights

- 5-domain demo support (robot, semiconductor, space, memory, semiforge/FYFab)
- Robot pilot-ready pack with skill memory and role reclassification
- Semiconductor pilot-ready pack with SemFab confidence explanation
- Recovery Compiler export boundary (candidate-only, no execution)
- Strict validation (53-point contract check)
- Public demo script with output index generation
- Public safety boundary documentation
- GitHub issue templates, PR template, CI workflow

## Safety boundary

HAL YieldOS does not control hardware, modify semiconductor recipes, run Recovery Compiler, generate recovery profiles, certify root cause, perform timing closure, or guarantee yield.

All outputs are candidate evidence for human review.
