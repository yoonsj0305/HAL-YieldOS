# HAL YieldOS

HAL YieldOSはAIモデルではありません。

HAL YieldOSは、ロボット、宇宙、半導体、メモリ、将来の機能歩留まりファブのデータを読み取り専用で分析し、エンジニアとAIが確認できる EvidencePack と Functional Passport を生成する機能歩留まりソフトウェアです。

---

## できること

- 産業データのスキーマ検証
- 状態スナップショット生成
- 機能歩留まり分析
- EvidencePack生成
- Functional Passport生成
- Decision Readiness生成
- 人間レビュー用の候補判断

## しないこと

- AIモデルとして判断を生成すること
- ロボット制御
- 衛星コマンド
- 半導体装置のrecipe変更
- 自動復旧
- root cause認証
- 安全認証
- 歩留まり保証

---

## 製品ポジション

HAL YieldOSは AI-ready Functional Yield Evidence Layer（機能歩留まり基盤産業証拠ソフトウェア）です。

AIは話す。YieldOSは証拠を作る。Forgeは覚える。人間が決定する。

---

## 対応ドメイン

| ドメイン | 説明 |
|----------|------|
| ロボット | 関節テレメトリ分析、技術者スキルメモリ |
| 宇宙 | 衛星電力・通信・姿勢異常分析 |
| 半導体 | ツールログ、ウエーハマップ、計測値分析 |
| メモリ | NANDフラッシュブロック健全性・機能再分類 |
| SemiForge | 欠陥耐性コンピューティングシミュレーション |
| FYFab Seed | 機能歩留まりファブシード（シミュレーション専用） |

---

## 安全境界

- ハードウェア実行なし
- 候補分析のみ
- 人間レビュー必須
- root cause認証なし
- 安全認証なし
